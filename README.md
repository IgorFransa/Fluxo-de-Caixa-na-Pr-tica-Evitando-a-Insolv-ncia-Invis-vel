# Caderno temático (esqueleto) — Fluxo de Caixa e Capital de Giro (Comércio)

> **Objetivo**: servir como *esqueleto completo* para você recriar o desafio no NotebookLM, com curadoria de 3–5 fontes, objetivos, perguntas estratégicas, testes de prompts (com variações e log), resumos, glossário e miniguia final.

## Como usar (rápido)

1. Abra esta pasta no **VS Code**.
2. Leia `ROADMAP.md` e siga a sequência (00 → 07).
3. No NotebookLM:
   - Crie um Notebook com o tema.
   - Faça upload/adicione as fontes listadas em `00_Fontes/links/FONTES.md`.
   - Use os prompts em `07_Miniguia_Prompts/PROMPTS_REUTILIZAVEIS.md`.
   - Registre as execuções no modelo `04_Testes_Prompts/TEMPLATE_LOG.txt`.
4. Ao final, copie o que você produziu no NotebookLM para os arquivos em `05_Resumos`, `06_Glossario_Formulas` e `07_Miniguia_Prompts`.

## Padrões do entregável

- **Idioma/encoding**: tudo em **UTF-8**.
- **Evidência**: o desafio pede **descrever** (não precisa print). Use os logs.

## Estrutura de pastas

- `00_Fontes/` → links e critérios de curadoria
- `01_Objetivos/` → objetivos de estudo
- `02_Perguntas/` → perguntas estratégicas
- `03_Case/` → situação-problema (comércio)
- `04_Testes_Prompts/` → logs (com variações)
- `05_Resumos/` → resumos por fonte
- `06_Glossario_Formulas/` → glossário + fórmulas
- `07_Miniguia_Prompts/` → miniguia final + biblioteca de prompts
- `templates/` → modelos para reaproveitar
